# Dataset Download Links

The following datasets were used in this project.

## 1. LIAR Dataset
Short political statements labeled for truthfulness.

Link:
https://www.cs.ucsb.edu/~william/data/liar_dataset.zip


## 2. FEVER Dataset
Fact Extraction and Verification dataset.

Link:
https://fever.ai/dataset/fever.html


## 3. SciFact Dataset
Scientific claim verification dataset.

Link:
https://github.com/allenai/scifact


## 4. FaceForensics++ (Deepfake Dataset)
Used for training the Xception deepfake detector.

Link:
https://github.com/ondyari/FaceForensics
